# Site Institucional GigaPet Resort 

Site desenvolvido para a rede de creches para cachorros GigaPet. O site conta com um CMS 
custmizado para a gestão de conteúdo.


## Funcionalidades

- Site Responsivo
- CMS Integrado (CMS próprio)
- Paiinel de controle
